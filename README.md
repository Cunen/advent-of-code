# Advent Of Code

https://adventofcode.com/

# Run App

```bash
npm install && npm start
```

# Creating solutions

- Create a new solution (exportable function)
- Add your solution to /aoc/challenges

```json
{
    year: 202X,
    day: X,
    fn: yourChallengeFn,
    part1: Perf.Ultra,
    part2: Perf.Ultra,
}
```

- Performance is automatically monitored, add part1/part2 solution performance after running succesfully
